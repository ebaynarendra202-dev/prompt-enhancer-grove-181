# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [1.2.0] — 2026-03-19

### Added
- **History Search Bar** — Full-text keyword search across original and improved prompts in history
- **History Category & Complexity Filters** — Filter prompt history by auto-detected category and complexity level
- **Color-coded History Badges** — Category and complexity badges displayed on each history item

## [1.1.0] — 2026-03-18

### Added
- **Coaching Tip Analytics** — Application/rejection rates, priority distribution, and engagement breakdown
- **Most Applied Tips Leaderboard** — Ranked view of which coaching tip categories are most valuable
- **Coaching Tip Trend Charts** — Volume and category breakdown of tip interactions over time
- **Analytics Trend Charts** — Time-series visualizations for prompts, categories, models, and templates
- **Comparison Badges** — Period-over-period percentage change indicators on analytics cards
- **Filterable Date Ranges** — Filter analytics dashboard by custom time periods
- **Milestone Notifications** — Toast notifications celebrating usage milestones
- **Template Favorites** — Bookmark built-in and custom templates for quick access
- **Template Usage Tracking** — Track which templates are used most frequently
- **AI-Powered Template Recommendations** — Get template suggestions based on prompt content
- **Prompt Sharing** — Generate public shareable links with optional expiration and view counts
- **Shared Prompt Page** — Public page for viewing original vs improved prompts side by side
- **Natural Language Input** — Describe what you need in plain language and get a structured prompt
- **Prompt Chains** — Multi-step prompt workflows with AI-suggested chains
- **Safety Analysis** — Prompt safety scoring with risk category identification
- **Auto-Categorization** — Automatic topic, complexity, and tag detection after each improvement
- **Data Export/Import** — Backup and restore prompt history as JSON

## [1.0.0] — 2026-03-17

### Added
- **AI-Powered Prompt Improvement** — Rewrite and enhance prompts with model-specific optimizations
- **Multi-Model Support** — GPT-4, GPT-3.5 Turbo, Claude 3 (Opus/Sonnet/Haiku), Gemini Pro, Llama 2
- **Quality Scoring** — Real-time quality analysis with actionable breakdown
- **Prompt Coaching** — Contextual real-time tips with one-click apply
- **Prompt Diff View** — Side-by-side comparison with highlighted changes
- **Version History** — Browse, restore, and manage previous prompt iterations
- **A/B Comparison** — Generate Clarity, Detail, and Creativity variants with trade-off analysis
- **Template Library** — Built-in and custom templates with category support
- **Analytics Dashboard** — Usage statistics, model breakdown, copy/favorite rates
- **User Authentication** — Email-based signup/login with session persistence
- **User Profiles** — Display name, bio, and avatar customization
- **Backup Codes** — Two-factor recovery code support
- **Dark/Light Mode** — System-aware theme toggle with manual override
